TAGS - HTML

Title, Strong, i, input, img

<Title> = Titulo da Pagina (Refencia ao Titulo da aba da pagina)

<Strong> = Se colocado essa tag dentro de um texto deixa ele em Negrito

Exemplo da Tag:

<strong>Texto em Negrito</strong>

Resultado:

Texto em Negrito

<i> = Coloca o Texto em Italico

Exemplo da Tag:

<i>Texto em Italico</i>

Resultado:

Texto em Italico

<input /> = Coloca um caixa de texto

Exemplo da Tag:

<p> input: <input type="text" placeholder="Insira o texto aqui =D" /></p>

OBS: Aqui usei o <p> para colocar um texto no inicio do Input

Resultado:


Input:

<img /> = Com essa tag conseguimos colocar imagens seja da internet ou do nosso Armazenamento Local.

Exemplo da Tag:

<img width="200" src="https://blog.sesisenai.org.br/wp-content/uploads/2021/09/shutterstock_329205053-scaled.jpg" />

Resultado:

Atributos Tags - HTML

id, style, class, type, src

<id> = atributo que adiciona um ID a um item do HTML

Exemplo do atributo:

<p id="label">Texto Label</p>

Resultado:

Texto Label

OBS: O Id não serve só para adicionar um ID mas também para localizar o item no HTML cujo o ID seja oque esta setado nele.

<style> = atributo que adiciona um "ESTILO" a um item do HTML como cor, largura, altura, etc...

Exemplo do atributo:

<p style="color:green">Texto Label Style</p>

Resultado:

Texto Label Style

<class> = atributo que adiciona uma "CLASSE" a um item do HTML como cor, largura, altura (usado com CSS Declaro no HTML)

Exemplo do atributo:

CSS:

<style>.border2{
border-width: 3px;
border-style: solid;
border-color: green;
}
</style>

HTML:

<p class="border2">Texto Label Class</p>

Resultado:

Texto Label Class

<type> = atributo que adiciona uma "TIPO" a um item do HTML como tipo texto, numero, cor, etc...

Exemplo do atributo:

<p>Selecione a Cor: <input type="color" /></p>

OBS: Aqui usei o <p> para colocar um texto no inicio do Input

Resultado:

Selecione a Cor:

<scr> = atributo que adiciona uma "SOURCE" a um item do HTML do tipo caminho - Local ou na WEB (mais comunmente usado em imagens)...

Exemplo do atributo:

<img src="https://www.projetodraft.com/wp-content/uploads/2019/12/digital-innovation-one.jpg" />

Resultado:

Tags para TEXTO - HTML

(h1 até h6), p, mark, small, i, u, strong, sup, blockquote, sub

<h1> = tag de cabeçalho em diferentes niveis do h1 até o h6 diminuindo em ordem decrescente.

Exemplo da TAG:

<h1>Cabeçalho nível 1</h1>
<h2>Cabeçalho nível 2</h2>
<h3>Cabeçalho nível 3</h3>
<h4>Cabeçalho nível 4</h4>
<h5>Cabeçalho nível 5</h5>
<h6>Cabeçalho nível 6</h6>

Resultado:

Cabeçalho nível 1

Cabeçalho nível 2

Cabeçalho nível 3

Cabeçalho nível 4
Cabeçalho nível 5
Cabeçalho nível 6

<p> = tag que representa um paragrafo.

Exemplo da TAG:

<h2>Cabeçalho</h2>
<p>paragrafo</p>

Resultado:

Cabeçalho

paragrafo

<mark> = tag serve como um "MARCADOR".

Exemplo da TAG:

<h2>Titulo de <mark>Exemplo</mark></h2>

Resultado:

Texto de Exemplo

<small> = tag serve como um sub-titulo.

Exemplo da TAG:

<h2>Titulo de Exemplo</h2><small>SMALL TEXT</small>

Resultado:

Texto de Exemplo

SMALL TEXT

<i> = Coloca o Texto em Italico.

Exemplo da Tag:

<i>Texto em Italico</i>

Resultado:

Texto em Italico

<u> = Coloca o Texto em Sublinhado/Underline.

Exemplo da Tag:

<u>Texto em Sublinhado</u>

Resultado:

Texto em Sublinhado

<Strong> = Se colocado essa tag dentro de um texto deixa ele em Negrito

Exemplo da Tag:

<strong>Texto em Negrito</strong>

Resultado:

Texto em Negrito

<sup> = Essa tag deixa o texto um pouco elevado (mais utilizado em calculos matematicos)

Exemplo da Tag:

<strong>Conta: 2 <sup>4</sup> = 16</strong>

Resultado:

Conta: 2 4 = 16

<blockquote> = Essa tag deixa o texto dentro de uma outra seção de Texto.

Exemplo da Tag:

<p>Texto inicial</p>
<blockquote>Texto blockquote</blockquote>

Resultado:

Texto inicial

Texto blockquote

<sub> = Essa tag deixa o texto abaixo da linha (mais utilizado em calculos matematicos)

Exemplo da Tag:

<strong>H<sub>2</sub>O</strong>

Resultado:

H2O

Listas ordenadas & Listas Não ordenadas - HTML

ol, ul, li

<ol> = tag de lista ordenada ordenando numericamente em ordem crescente.

<li> = tag de item da lista "list item".

Exemplo da TAG:

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Resultado:

  1. Item 1
  2. Item 2
  3. Item 3

<ul> = tag de lista não ordenada.

Exemplo da TAG:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Resultado:

  • Item 1
  • Item 2
  • Item 3

***BONUS***

É possivel usar a tag de lista ordenada e não ordenada com sub-items.

Exemplo da TAG:

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
        <li>Item 4
            <ul>
                <li>Item 4.1</li>
                <li>Item 4.2</li>
            </ul>
        </li>
</ul>

Resultado:

  • Item 1
  • Item 2
  • Item 3
  • Item 4
    • Item 4.1
    • Item 4.2

Links - HTML

a, href, title, target

<a> = tag de "ANCORA" (normalmente usada quando sinalizada como links).

<href> = atributo "REFERENCIA" (normalmente usada para links).

<title> = atributo quando usado dentro de alguma tag clicavel, abrira um "CAIXINHA DE TEXTO DICA".

<target> = atributo "ALVO" quando colocado em uma tag clicavel. (Muito utilizado atualmente para quando clicar no item html abrir uma nova pagina ou usar a mesma).

Exemplo da Tag:

<a> href="http://dio.me" title="Clique aqui para saber mais" target="_blank">Dio.me</a>

Resultado:

Dio.me

OBS: Se você passar o Mouse por cima e esperar um pouco a caixinha de Texto aparece =D

Explicação dos Atributos:

href: Pode ser utilizado tanto para sites como arquivos locais
(quando locais basta apenas digitar o nome do arquivo se estiver na mesma pasta, caso contrario digite o caminho do arquivo)

title: Atributo quando usado dentro de alguma tag clicavel, abrira um "CAIXINHA DE TEXTO DICA".

target: Para abrir em uma nova pagina utilize entre as apas _blank, caso queira usar a mesma aba para entrar no site ou arquivo local_self

BONUS ++

Sobre as tags (font, del, p, abbr)

<font> = tag para setar o Tamanho da Fonte.

Exemplo da Tag:

<p>Testando a tag <font size="5">TAMANHO DA FONTE</font></p>

Resultado:

Testando a tag TAMANHO DA FONTE

<del> = tag para Deletar um Texto de uma frase/ficar riscado.

Exemplo da Tag:

<p>www.google<del>.com</del></p>

Resultado:

www.google.com

<p> = tag que representa um paragrafo.

Exemplo da TAG:

<h2>Cabeçalho</h2>
<p>paragrafo</p>

Resultado:

Cabeçalho

paragrafo

<abbr> = tag que representa uma abreviação.

Exemplo da TAG:

<abbr title="Digital Innovation One">DIO</abbr>

Resultado:

DIO

OBS: Se passar o mouse por cima do texto vai ver o Texto Completo =D
